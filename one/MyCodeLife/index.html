从玻璃生产线员工到程序员需要几个步骤

发表于2021-08-02|更新于2021-08-04|随笔
|阅读量:

建材学院

1.考了vf 首次接触编程
2.参加了北大青鸟的线下公开课 对编程产生了兴趣

沙河金鑫玻璃厂员工

1.员工 有没有人知道sogoyo.cn这个网站，当时有很多破解游戏和汉化游戏。当时使用手机进行汉化游戏玩。
2.班长 有没有人知道sogoyo.cn这个网站，当时有很多破解游戏和汉化游戏。当时使用手机进行汉化游戏玩。

唐图科技游戏客服运营 双城OL 玄境OL 星弹堂OL

1.在这里了解到程序员的工资比其他非领导岗位要高。
2.做了一些对游戏的理解、Ui布局、功能策划、运营活动数据分析准备
3.每天下班看c语言教程 学习了当时的清华使用的编程书籍和算法书籍
4.准备换工作

迷你酷联游戏策划 士兵联盟

1.执行策划
2.数值策划 包括经济数值 战斗数值 期间学了些excel 以及vb开发

创业开发 士兵联盟

1.接触到cocos2d-3.0 从0开发士兵联盟。

睡神飞程序员 coco2d unity 太极迷阵 十字迷阵 魔性建筑工 十步万度 Idle Bubbles 以及参与我飞刀玩得贼6 123Bom ToB

1.Windows环境，cocos2d-3.0开发android游戏接触到Androistudio，搞定从开发到发布Android应用到谷歌商店以及taptap等平台全流程
2.Windows环境下搭建mac环境接触到xcode
3.购买了appstore开发者，走通了从windows环境下cocos开发到xcode出ios应用到上传发布全流程
4.参加了24小时开发者大会，接触到Unity
5.Unity从开发exe、Android、Ios到发布全流程
6.常用的SDK接入（taptap、admob、头条、yomob、友盟、微信登录、微信支付、支付宝支付、等等）
7.接触到Github gitlab

熊孩子程序员 unity node服务器 TensUp 我爱消方块 黄金球球 欢乐水族馆

1.更熟练的Unity开发
2.更完善的开发思想
3.更优化的开发习惯
4.15天从阿里云购买到部署node express mogodb Windows linux环境搭建到正式游戏服务器
5.服务器版署实名认证、服务器苹果微信支付宝支付
5.Hexo博客停不下来了
