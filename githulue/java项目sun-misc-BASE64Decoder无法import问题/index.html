java项目sun.misc.BASE64Decoder无法import问题

发表于2023-01-11|更新于2023-01-11|随笔
|阅读量:

本地jdk1.8.0_311,
右键项目 Properties ->Java Build Path->Libraries->找到JRE System Libray[JavaSE-1.8]->右侧Edit按钮->选择Alternate JRE->点击Installed FREs按钮
->右边点击Add按钮->选择Standard VM->点击下方next按钮->JRE home中填jdk路径，或者点击Directory找到jdk路径比如：D:\Program Files\Java\jdk1.8.0_311-》
然后点击Finish按钮-》然后勾选刚刚加的jdk->点击下方Apply and close->这时候回到了之前打开的界面，Systemlibray，Execution environment 出现JavaSE-1.8(jdk1.8.0_311)
类似就可以了-》点击完成->点击apply
